    Desarrollo y diseño web

    HTML

    HTML es la base fundamental de la construcción de páginas web y permite definir la estructura y el contenido de una manera que los navegadores web pueden interpretar y mostrar.

    CSS

    CSS es esencial para dar estilo y diseño a las páginas web, lo que mejora su apariencia y usabilidad.

    JavaScript

    JavaScript es una parte fundamental del desarrollo web moderno y es compatible con la mayoría de los navegadores web, lo que lo convierte en una herramienta poderosa para la creación de aplicaciones web y sitios web interactivos.

    Python

    Python es apreciado por su enfoque en la legibilidad del código, lo que facilita la colaboración y el mantenimiento del software. Es un lenguaje poderoso y versátil que se ha convertido en una de las opciones más populares en el mundo de la programación.

    Poo

    La Programación Orientada a Objetos es una metodología de diseño de software que se utiliza para modelar problemas del mundo real y organizar el código de manera más clara y mantenible. Se utiliza en una amplia variedad de lenguajes de programación, como Java, C++, Python, C#, y muchos otros, y es especialmente útil en el desarrollo de aplicaciones complejas y sistemas de software.



    Desarrollo de videojuegos

    Unity

    Unity es una plataforma poderosa y versátil para el desarrollo de juegos y aplicaciones interactivas en una variedad de plataformas. Su facilidad de uso, la comunidad activa y la amplia gama de recursos hacen que sea una elección popular para desarrolladores de juegos y aplicaciones en todo el mundo.

    Unreal engine

    Unreal Engine es apreciado por su capacidad para crear juegos de alta calidad y experiencias visuales impresionantes. Es utilizado por desarrolladores de videojuegos, cineastas y profesionales de diversas industrias para crear contenido interactivo y experiencias visuales realistas.
